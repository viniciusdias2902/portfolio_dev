# Portfólio Pessoal

Um site de portfólio responsivo desenvolvido para praticar HTML e CSS, apresentando seções de apresentação, habilidades, projetos e contato.

## Sobre o Projeto

Site de portfólio pessoal de uma página com design moderno e responsivo. O projeto inclui navegação fixa, seção hero com imagem de fundo, apresentação pessoal, showcase de habilidades técnicas, galeria de projetos e formulário de contato.

## Seções

- **Navegação**: Menu fixo no topo com links para as seções
- **Showcase**: Apresentação inicial com imagem de fundo em tela cheia
- **Sobre Mim**: Seção com foto e descrição pessoal
- **Habilidades**: Cards com as principais tecnologias dominadas
- **Projetos**: Galeria de projetos com imagens e descrições alternadas
- **Contato**: Formulário para envio de mensagens
- **Footer**: Links para redes sociais e copyright

## Características

- Design responsivo com breakpoint em 1024px
- Layout em flexbox para organização dos elementos
- Navegação oculta em dispositivos móveis
- Formulário de contato estilizado
- Efeitos hover em elementos interativos
- Background overlay na seção hero
- Font Awesome para ícones sociais

## Tecnologias

- HTML5
- CSS3 (Flexbox, Media Queries, Transitions)
- Google Fonts (Poppins)
- Font Awesome

## Como Usar

Clone o repositório e abra o arquivo `index.html` em seu navegador.

```bash
git clone https://github.com/seu-usuario/portfolio.git
cd portfolio
```

## Aprendizados

Prática de desenvolvimento front-end focada em layouts responsivos, organização de código CSS, trabalho com formulários e integração de bibliotecas externas.
